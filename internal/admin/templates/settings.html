Configuration Guide

guest_otp_enabled

Toggle verifikasi OTP untuk guest login. ON = email + kode + OTP. OFF = email + kode saja (tanpa verifikasi email).

max_subscribers

Maksimal subscriber aktif secara bersamaan. Jika tercapai, user baru tidak bisa subscribe.

max_active_guests

Maksimal guest session aktif. Limit ini untuk mengontrol beban server dari akun trial.

quota_warning_pct

Persentase kuota dimana admin akan menerima notifikasi warning. Default: 80%.

guest_priority_mode

Jika ON, guest akan diblokir total saat subscriber kuota >90%. Set OFF untuk mode normal.
